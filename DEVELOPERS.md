# Contribuir a Parallel Zero Framework
## Introducción
Gracias por considerar contribuir a Parallel Zero. Este framework busca ser una herramienta simple pero poderosa para desarrolladores PHP. Su diseño minimalista está pensado para proyectos de todos los tamaños.

## Estructura del Proyecto
src/: Código fuente principal del framework
Core/: Funcionalidades base
Database/: Capa de abstracción de bases de datos
Routing/: Sistema de enrutamiento
app/: Donde reside tu aplicación
Controllers/: Controladores de tu aplicación
Models/: Modelos de tu aplicación
## Cómo Contribuir
Forkear el repositorio y clonarlo localmente
Crear una nueva rama: git checkout -b my-feature-branch
Hacer cambios y testear
Enviar un Pull Request con una buena descripción de los cambios
## Directrices de Código
Seguir PSR-1, PSR-4 y PSR-12 para estilo de código
Escribir pruebas para nuevas funcionalidades
Comentar el código y mantener la documentación actualizada
